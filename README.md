# 🧮 Calculadora Web 

Projeto de **calculadora online** desenvolvido com **HTML5**, **CSS3** e **JavaScript (Vanilla JS)**.  
A aplicação permite realizar operações matemáticas básicas em uma interface simples, responsiva e com visual moderno (incluindo imagem de fundo).

---

## ✅ Funcionalidades

- Operações básicas: **adição (+), subtração (-), multiplicação (×) e divisão (÷)**
- **Limpar** (C/AC) e **apagar último dígito** (se implementado)
- Suporte a **números decimais**
- Interface amigável e responsiva
- Estilização com CSS e fundo personalizado

---

## 🧰 Tecnologias

- **HTML5**
- **CSS3**
- **JavaScript (Vanilla JS)**

---

## ▶️ Como executar

### Opção 1 — Abrir direto no navegador
1. Faça o download/clone do projeto
2. Abra o arquivo `index.html` no navegador

### Opção 2 — Live Server (VS Code)
1. Instale a extensão **Live Server**
2. Clique com o botão direito no `index.html` → **Open with Live Server**
